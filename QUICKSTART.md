# Quick Start

## First Time Only

Clone the repo and run setup:

```bash
git clone https://github.com/rohunj/claude-build-workflow.git
cd claude-build-workflow

./one-time-setup.sh
./setup-notifications.sh
```

Then install the **ntfy** app on your phone (free) and subscribe to your topic.

---

## Every New Project

### Step 1: Start

```bash
cd /path/to/claude-build-workflow && claude
```

### Step 2: Say This

```
Run the workflow
```

### Step 3: Answer Questions

Claude guides you through everything:
- What you're building
- Who it's for
- Technical decisions
- Etc.

### Step 4: Start the Build

When Claude asks "Ready to start the build?", say **"Start now"**

### Step 5: Done

- Close your laptop
- Build runs automatically
- Get notified on your phone when it's done
- Check from phone anytime at github.com/codespaces
