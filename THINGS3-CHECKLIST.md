# Things 3: Build a New Project

## First Time Only (One-Time Setup)
- [ ] Clone repo: `git clone https://github.com/rohunj/claude-build-workflow.git`
- [ ] Run: `cd claude-build-workflow && ./one-time-setup.sh`
- [ ] Run: `./setup-notifications.sh`
- [ ] Install ntfy app on phone and subscribe to your topic

---

## Every New Project
- [ ] Terminal: `cd /path/to/claude-build-workflow && claude`
- [ ] Say: "Run the workflow"
- [ ] Answer Claude's questions
- [ ] Say "Start now" when ready
- [ ] Close laptop - get notified when done

---

## Check from Phone (if needed)
- [ ] Go to github.com/codespaces
- [ ] Open your project's codespace
- [ ] View terminal for progress
